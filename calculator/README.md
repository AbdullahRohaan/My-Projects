# Python Calculator

## Overview
This project is a menu-driven calculator application developed using Python. It allows users to perform basic and advanced mathematical operations through a simple and user-friendly command-line interface. The program includes proper input validation and error handling to ensure smooth execution.

## Features
- Addition, Subtraction, Multiplication, and Division
- Power and Square Root calculations
- Handles invalid numeric inputs gracefully
- Prevents division by zero errors
- Modular and well-structured code

## Tech Stack
- Python
- Standard `math` library

## How to Run
1. Open the project folder in VS Code or any code editor.
2. Make sure Python 3 is installed on your system.
3. Run the following command in the terminal:
   ```bash
   python calculator.py
